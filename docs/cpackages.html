Other related Packages

 

MUSIC

The MUSIC fluid dynamical evolution package is the default for JETSCAPE and can be independently downloaded at http://www.physics.mcgill.ca/music/

 

CLVisc

CCNU-LBNL viscous 3+1 D hydro code is a fully parallelized hydro evolution and Cooper-Fry particlization code using OpenCL.

It can be run on heterogeneous computer devices (CPU, GPU, FPGA and Intel Phi).

It can be downloaded at gitlab.com/snowhitiger/PyVisc
